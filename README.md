# baby-cry-detector
mendeteksi dan mengklasifikasikan tangisan bayi untuk mengetahui apa yang diinginkannya, seperti rasa lapar (hungry), ketidaknyamanan (discomfort), atau kelelahan (tired). Proyek ini menggunakan sistem yang memanfaatkan teknik pemrosesan audio dan deep learning model untuk mencapai klasifikasi yang akurat.
